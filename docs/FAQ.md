# FAQ

Q: Como funciona?
A: Você escolhe um template, responde 5 perguntas, geramos e fazemos deploy automático.

Q: Quanto tempo leva?
A: Geração ~10 segundos, deploy ~60 segundos, total ~90 segundos.

Q: Posso cancelar?
A: Sim, a qualquer momento. Cobrança mensal via Stripe.

Q: Quais integrações estão disponíveis?
A: Email, Slack e Webhooks (varia por template).

Q: Self-hosted está disponível?
A: Sim, via Docker (pagamento único).

Q: Como monitorar erros e performance?
A: Configuramos Sentry para server e frontend; métricas básicas via logs. Prometheus + Grafana planejado.
