# Contributing

Thank you for considering contributing to Build-O-Matic!

- Read docs/AI_INSTRUCTIONS.md and docs/ARCHITECTURE.md
- Keep modules small, documented, and tested
- Prefer adding tests before large changes
- Follow TypeScript strict mode
- Do not hardcode secrets; use environment variables
- Use feature/integration block conventions in templates:
  - FEATURE:<id> START/END
  - INTEGRATION:<id> START/END
